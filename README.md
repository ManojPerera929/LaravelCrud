### Laravel Simple Crud


### Requirements

- php 8.1
- Laravel 9
- Mysql




### SET FOLLOWING ENV

```dotenv


#Database Config
DB_CONNECTION = 
DB_HOST = 
DB_PORT=
DB_DATABASE =
DB_USERNAME=
DB_PASSWORD=








```

### SETUP
- composer install

## Commands
- <code>php artisan migrate</code> - Run All Migrations
